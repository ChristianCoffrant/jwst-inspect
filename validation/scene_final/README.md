# Week 8 Final Scene Gate

This directory stores small metadata files for the Workstream 1 Week 8 scene
freeze. Render PNGs and contact sheets are generated artifacts and remain under
`validation/renders/week8_final/`, which is ignored by Git.

The hard final gate is not satisfied until `week8_final_render_gate.yaml`
records a successful synced x090-class Vast.ai/Isaac Sim run and matching run
registry row.
