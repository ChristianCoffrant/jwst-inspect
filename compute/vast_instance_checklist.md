# Vast.ai Instance Checklist

Before starting paid work:

- [ ] GPU is RTX 4090, RTX 5090, or approved RTX workstation class.
- [ ] VRAM is recorded.
- [ ] System RAM is at least 64 GB preferred.
- [ ] Disk space is sufficient for the planned run.
- [ ] `nvidia-smi` works.
- [ ] Isaac Sim or selected container starts.
- [ ] A tiny render or smoke test completes.
- [ ] Repo commit hash is recorded.
- [ ] Storage sync path is configured.
- [ ] Shutdown plan is explicit.

After the run:

- [ ] Artifacts synced.
- [ ] Run registry updated.
- [ ] Cost log updated.
- [ ] Instance stopped.
